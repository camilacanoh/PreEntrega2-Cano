El editor de Canva tiene varias secciones: la navegación principal, el panel lateral, la barra de herramientas, el lienzo y los controles de página. Para enfocarte en una sección y conocer más de qué se trata, usá el tabulador. Para ir al lienzo, presioná "Ctrl+F2".
La navegación principal contiene botones con opciones para trabajar sobre el documento. Para pasar de un botón a otro, usá las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usá la tecla Enter. Para ir al lienzo, presioná "Ctrl+F2".

El título del diseño tiene el nombre de tu diseño. Para editar el título del diseño, presioná la tecla Enter. Para dejar de editar el título del diseño, presioná la tecla Escape.
La barra de herramientas contiene botones para editar el elemento seleccionado. Para pasar de un botón a otro, usá las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usá la tecla Enter. Para ir al lienzo, presioná "Ctrl+F2".
El lienzo es donde está tu diseño. Para ir a la página, presioná la tecla Enter. Para ir a grupos o elementos en la página, volvé a presionar la tecla Enter. Presioná el tabulador para ir a la siguiente página, al siguiente grupo o al siguiente elemento. Para usar la barra de herramientas, usá la tecla “Ctrl+F1”. Si hay un elemento seleccionado, podés moverlo usando las teclas de flechas, podés rotarlo con la tecla “,” y “.” y podés cambiarle el tamaño con las teclas de flechas y la tecla “Ctrl”. Podés aumentar los incrementos de todos estos atajos con la tecla "Mayús".
